## explain/desc

#### 语法

> explain + SQL 语句（select 语句）
>
> desc + SQL 语句（select 语句）
> 
>> 


```sql
## 例如： explain select * from t3 where id=1;
+----+-------------+-------+-------+-------------------+---------+---------+-------+------+-------+
| id | select_type | table | type  | possible_keys     | key     | key_len | ref   | rows | Extra |
+----+-------------+-------+-------+-------------------+---------+---------+-------+------+-------+
```

explain执行计划中包含的信息如下：

- id:  查询序列号
- select_type: 查询类型
- table: 表名或者别名
- partitions: 匹配的分区
- type: 访问类型
● possible_keys: 可能用到的索引
● key: 实际用到的索引
● key_len: 索引长度
● ref: 与索引比较的列
● rows: 估算的行数
● filtered: 按表条件筛选的行百分比
● Extra: 额外信息
